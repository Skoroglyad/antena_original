A) HTML Structure - top

  1. The template is divided to sections each section has id that refers to its content like (section id = "about").
  2. Template content such as text, images, icons and buttons is inside container div (div class = "container").
  3. The container div holds the section structure.
  4. The "title" div is holding a heading title of each section.
  5. The row div (div class = "row") and divs which have classes like this (div class = "col-md-6") are for responsive "Bootstrap Part".
  6. The structure of the section is like (section -> container).

C) JavaScript - top

This template imports 5 Javascript files.

  2. Bootstrap JS
  3. custom.js (The Custom JavaScript File)
  4. Lightbox JS
  5. Carousel JS
